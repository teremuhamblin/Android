# 📂 Exploration du système de fichiers Android
### _The MadDoG Android System — Tools Module_

Android utilise un système de fichiers dérivé de Linux.

---

## 📁 1. Emplacements importants

| Chemin | Rôle |
|--------|------|
| `/system` | Framework + apps système |
| `/vendor` | Drivers constructeur |
| `/data` | Données utilisateur |
| `/sdcard` | Stockage accessible |
| `/proc` | Infos kernel |
| `/sys` | Interfaces matériel |
