# 🔐 File-Based Encryption (FBE)
### _The MadDoG Android System — Security Module_

FBE est le système de chiffrement moderne d’Android.

---

## 🧩 1. Principe

Chaque fichier est chiffré individuellement.  
Deux espaces existent :

- **DE** — Device Encrypted  
- **CE** — Credential Encrypted  

---

## 🔐 2. Avantages

- Sécurité renforcée  
- Déverrouillage plus rapide  
- Isolation par utilisateur  

---

## 🧠 3. Fonctionnement

- Clés stockées dans `metadata`  
- Déverrouillage via PIN / biométrie  
- Intégration avec SELinux
